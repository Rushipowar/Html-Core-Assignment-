MEAN Stack

The MEAN Stack is full-stack Javascript framework used for buildig dynamic web applications.

  • MongoDB for the NoSQL database.
  • Express.js for the web applications framework.
  • Angular for the frontend framework.
  • Node.js for server-side Javascript runtime.
MERN Stack

The MERN Stack is full-stack Javascript framework designed single-page applications(SPAs).

  • MongoDB for the NoSQL database.
  • Express.js for the web applications framework.
  • React for building user interface.
  • Node.js for server-side Javascript runtime.
LAMP Stack

The LAMP Stack is a classic web development stack known for its versatility.

  • Linux as the oprating system.
  • Apache as web server.
  • MySQL for relational database managment system.
  • PHP for server-side scripting.
JAMStack

IT is modern architecture for building fast and secure websites.

  • JavaScript for dynamic functionality.
  • APIs for data retrieval and integration.
  • Markdown content authoring.
PERN Stack

The PERN Stack is a tech stack idea for creating data-drive applications.

  • PostgreSQL for the database.
  • Express.js for the web applications framework.
  • React for building user interface.
  • Node.js for server-side Javascript runtime.

List Of Tags

  1. html: root element of HTML document that encloses all other HTML elements.
  2. head: Contains meta-information about the document, such as title, character encoding, and linked resources like stylesheets.
  3. title: Specifies title of web page. displayed in browser title bar or tab and used search engine result.
  4. meta: Provides metadata about the document, includig charater, encoding, authorship, and page descriptions.
  5. link: Established link to external resources like stylesheet, icons and alternate version of document.
  6. script: Embeds or references JavaScript code that can add interactivity and functionality to the web page.
  7. body: Contains visible content of the web page including text, images, videos, links etc..
  8. div: A versitle container used for grouping and structuring content or applying CSS styles.
  9. h1-h6: Heading h1 for highest and h6 for lowest that define importance of text.
  10. p: Represent paragraph of text creating spacing and formatting around the enclosed content.
  11. a: Create hyperlink to another pages or resouces. Its allow to user navigate within or outside the website.
  12. img: Embeds images on webpage with attributr src, alt etc..
  13. li: Reprents list items within ul and ol.
  14. ul: For Unorder-List to create bullet-pointed lists.
  15. ol: Order-Lists to create numbred lists.
  16. table: Set up table structure for displaing data with rows and columns using tr,th,td elements.
  17. form: Encloses form elements like text fields, checkboxes and buttons used for user input and submission.
  18. input: Create input fields within a form such as text, password, radio buttons and checkboxes.
  19. label: Provide a label or description for form elements, improving accessibility and user experience.
  20. textarea: Define a multiline text input area, commonly used fot longer user input or comments.
